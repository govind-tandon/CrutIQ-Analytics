# CrustIQ-Analytics ( Interactive Dashboard using MS Excel, SQL, Power BI )

## Project Objective
The CrustIQ Store owner requested an annual sales report for 2025 to gain insights into customer behavior and identify opportunities to increase sales in 2026 and 2027.

# Questions (KPI's)
- Total Revenue
- Average Order Value
- Total Pizzas Sold
- Total Orders
- Average pizzas per order
- Number of pizzas sold by the total number of orders.
